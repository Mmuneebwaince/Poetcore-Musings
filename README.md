# LunaBelle — Demo Clothing Brand Website

This repository contains a simple, responsive static website for a female clothing brand named LunaBelle. It includes a Home page, a Shop listing, an About page, and a Contact page.

What you get
- Clean, responsive layout with a feminine aesthetic
- Lightweight shop grid with interactive "Add to cart" feedback
- Newsletter signup with basic validation
- Accessible navigation and semantic HTML

How to run
- Open index.html in your browser
- Or host with a static server of your choice
